K-Means Image Processing Algorithms

What it is

An implementation of K-Means clustering applied to image pixels. The algorithm iteratively groups pixels into color clusters, producing effects like color quantization (reducing an image to fewer colors), edge extraction between regions, and palette shuffling where colors get reassigned to different regions.

Blur preprocessing reduces noise for cleaner clustering results, and you can progressively increase the cluster count to recover detail.

How it works

K-Means treats each pixel as a data point in 3D color space (R, G, B). It picks k random cluster centers, assigns every pixel to the nearest center, then recalculates centers as the mean of their assigned pixels. Repeat until convergence. The result: every pixel gets snapped to its cluster's color.

From there you can extract edges (boundaries between different clusters), swap palettes, or animate the algorithm's progression to watch clusters form in real time.

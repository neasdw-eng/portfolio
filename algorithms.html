Algorithm Visualizer

Color-coded, step-by-step visualizations of classic algorithms — Dijkstra's shortest path, Graham Scan convex hull, and Quick Sort.

TypeScript React Algorithms Visualization

What it is

Interactive visualizations that animate algorithms step by step, using color coding to make each phase of the algorithm intuitive. Built with TypeScript and React, it covers graph algorithms, computational geometry, and sorting.

Dijkstra's Shortest Path

Green = visited nodes, purple = current edge being evaluated, blue = edges in final shortest path.

Graham Scan (Convex Hull)

Nodes sorted around anchor point, purple lines show the 3 nodes being evaluated for hull membership.

Quick Sort

Green = current pivot, red = section being sorted.

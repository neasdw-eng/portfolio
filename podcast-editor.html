Podcast Editor

A full-pipeline podcast editing app — upload raw footage, and it handles transcription, cleanup, editing, and export.

React 19 TypeScript FastAPI Whisper Claude AI WebSockets FFmpeg WaveSurfer.js

The projects dashboard — pick up where you left off or start a new episode.

What it is

A web-based podcast editor that takes raw video/audio recordings and runs them through an automated processing pipeline: transcription via Whisper, silence detection, AI-powered content flagging (via Claude), filler word detection, mouth noise cleanup, and noise reduction.

After processing, you get an interactive waveform editor where you can review, approve, or reject each detected issue — then export a polished MP4 or MP3. The whole thing runs locally, no cloud uploads needed.

Key Features

  • Automatic processing pipeline — Upload files and the backend handles: audio extraction, Whisper transcription, silence detection, AI content flagging, mouth noise detection, filler word detection, and noise reduction.
  • Waveform timeline editor — Visual waveform with color-coded segments (silence, AI-flagged, bleeps, mutes, manual cuts). Click to review, drag to adjust boundaries.
  • AI content flagging — Claude analyzes the transcript and flags potentially problematic segments with explanations, so you can review before publishing.
  • Word-level bleeping — Search the transcript and bleep specific words with a tone or mute, with precise word-level timing from Whisper.
  • Speaker diarization — Automatic speaker separation with color-coded tracks. Assign names and labels to each speaker.
  • Real-time progress — WebSocket connection streams processing status and export progress, including live video frame previews during render.
  • Multi-file support — Upload multiple MP4/MOV files that get concatenated in order, with 10MB chunked uploads for large files.

The Processing Pipeline

When you upload files, the backend kicks off a multi-stage pipeline. Each stage runs as a background task with real-time progress updates via WebSocket.

Upload Chunked upload (10MB) → file concatenation → MP4/MOV
↓
Extract FFmpeg audio extraction → WAV for processing
↓
Transcribe faster-whisper → word-level timestamps + speaker indices
↓
Detect Silence, mouth noise, filler words, padding phrases (parallel)
↓
AI Flag Claude reviews transcript → flags problematic content with reasons
↓
Edit Interactive waveform editor → approve/reject/adjust segments
↓
Export Noise reduction → normalization → FFmpeg render → MP4/MP3

Why I built it

I was recording podcast episodes and the editing workflow was painful — silence trimming, content review, bleeping, and export all required different tools. I wanted one app that could take raw footage and handle the entire pipeline, with AI doing the tedious parts (flagging content, detecting filler words) so I could focus on creative decisions.

The waveform editor is the centerpiece. Each detected issue shows up as a color-coded segment on the timeline — amber for silence, red for AI-flagged content, pink for bleeps. You review each one, approve or reject, adjust the boundaries if needed, and export. The whole flow from raw footage to finished episode happens in one place.

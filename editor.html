Speech-Based Code Editor

An Eclipse plugin that enables hands-free Java development through voice commands — built as a college team project at Brown.

Java Eclipse Plugin Google Speech API Team Project

What it is

A plugin for the Eclipse IDE that lets you write Java code using your voice. It listens in real-time via Google Speech API, converts speech to text, then maps that text to development actions — creating classes, writing methods, navigating files, all without touching the keyboard.

Built as a team project at Brown, it supports custom phrase configurations so you can define your own voice commands for specific coding patterns.

How it works

  • Real-time speech listening — Continuous audio capture streamed to Google Speech API for transcription.
  • Custom phrase detection — Configurable phrase mappings that translate spoken commands into IDE actions.
  • Text → Command pipeline — Speech text is parsed, matched against phrase configs, and dispatched as Eclipse editor commands.
  • Nearly hands-free workflow — Write entire Java programs, navigate code, and manage files through voice.

Online Chess

Multiplayer chess with real-time game sync via Firebase, move validation, and an AI opponent powered by the minimax algorithm.

React Firebase chess.js Minimax AI

What it is

An online multiplayer chess game built with React and Firebase. Create a game, share the invite link, and play in real-time with moves synced instantly via Firebase. Or play against an AI that uses the minimax algorithm to evaluate positions.

Key Features

  • Real-time multiplayer — Firebase handles game state sync so moves appear instantly for both players.
  • Full move validation — chess.js handles legal move detection, check/checkmate, and game state tracking.
  • AI opponent — Minimax algorithm with a piece-count evaluation function. Not grandmaster-level, but it'll catch your blunders.
  • Game creation flow — Create a game, get a shareable link, opponent joins and you're playing.
